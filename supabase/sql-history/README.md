# SQL History

Folder ini adalah catatan SQL manual untuk Supabase, bukan migration otomatis.

Sebelum mengeksekusi file:

1. Baca header dan ketergantungannya.
2. Pastikan perubahan tersebut belum diterapkan.
3. Backup data yang terdampak.
4. Jalankan melalui Supabase SQL Editor.
5. Verifikasi hasil query dan policy RLS setelah eksekusi.

File lama tetap dipertahankan agar riwayat perubahan tidak hilang.
